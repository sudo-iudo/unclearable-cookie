#### Steps to reproduce

1. Open the [page](https://sudo-iudo.github.io/unclearable-cookie) in Google Chrome, Safari, or Firefox
2. Clear the cookies using the popup from the address bar
3. Reload the page

#### Disclaimer

This method works in Chrome, Safari, and Firefox.  
It does not work if the cookies are cleared while the tab is closed. However, it still bypasses the most common cookie-clearing flow in the UI.
